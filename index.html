Install on Linux

  1. Download latest release:
    wget https://github.com/SpidermanTotro/nebulultimate/releases/latest/download/nebula-ultimate.AppImage
  2. Make it executable & run:
    chmod +x nebula-ultimate.AppImage
    ./nebula-ultimate.AppImage
  3. For .deb or .tar.gz:
    wget https://github.com/SpidermanTotro/nebulultimate/releases/latest/download/nebula-ultimate.deb
    sudo dpkg -i nebula-ultimate.deb
  4. For source: git clone https://github.com/SpidermanTotro/nebulultimate.git
    Follow README.md instructions
